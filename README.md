### 主要相关技术：
- HTML、CSS、JQuery、JavaScript、Ajax、ES6、webpack
- 使用vw为长度单位,适应多种手机端页面
- 通过AJAX获取JSON数据,将歌曲列表,图片以及其他信息展现到页面上,JSON数据是自己制作,只有十首歌曲,所以在搜索界面搜索其他歌曲显示没有结果.
- 歌曲播放时歌词根据歌曲的时间进行滚动
